การเปรียบเทียบการรันโค้ดในภาษา Java, C++, Python

บทความนี้จะกล่าวถึงการเขียนโปรแกรมในภาษา Java, C++, และ Python โดยใช้ตัวอย่างโค้ดที่สร้างคลาส Animal, Dog, และ Cat จากนั้นเปรียบเทียบผลการรันในแต่ละภาษา

1. Java

โค้ดใน Java สร้างคลาส Animal, Dog, และ Cat โดยมีการ override เมธอด sound() ในคลาสลูก

Animal a = new Animal();
Dog d = new Dog();
Cat c = new Cat();

a.sound();
d.sound();
c.sound();

System.gc();  // garbage collection (calls destructor)

ผลรัน:

create animal
dog
create animal
cat
create animal
animal makes a sound
dog barks
cat meows
delete animal
delete dog
delete animal
delete cat
delete animal

ในผลลัพธ์นี้ สังเกตได้ว่าแต่ละคลาสจะพิมพ์ข้อความตาม constructor และเมธอด sound() ที่ถูก override เมื่อเรียกใช้งาน จากนั้นเมื่อ garbage collection ทำงาน destructors จะถูกเรียก

2. C++

โค้ดใน C++ ใช้ pointer เพื่อสร้างวัตถุของคลาส Animal, Dog, และ Cat และมีการใช้ delete เพื่อลบวัตถุ

Animal *a = new Animal();
Dog *d = new Dog();
Cat *c = new Cat();

a->sound();
d->sound();
c->sound();

delete a;
delete d;
delete c;

ผลรัน:

create animal
dog
create animal
cat
create animal
animal makes a sound
dog barks
cat meows
delete animal
delete dog
delete animal
delete cat
delete animal

การรันใน C++ ให้ผลลัพธ์คล้ายกับใน Java ต่างกันตรงที่ใช้ delete เพื่อลบวัตถุ ทำให้ destructors ถูกเรียกเพื่อทำความสะอาดหน่วยความจำ

3. Python

โค้ดใน Python ใช้คลาสและเมธอดเหมือนใน Java และ C++ โดยมีการใช้ del เพื่อลบวัตถุ

a = Animal()
d = Dog()
c = Cat()

a.sound()
d.sound()
c.sound()

del a
del d
del c

ผลรัน:

create animal
dog
create animal
cat
create animal
animal makes a sound
dog barks
cat meows
delete animal
delete dog
delete animal
delete cat
delete animal

ใน Python การลบวัตถุจะใช้ del ซึ่งจะทำให้เมธอด __del__() (destructor) ถูกเรียก ทำให้พิมพ์ข้อความเหมือนใน Java และ C++

สรุปการเปรียบเทียบผลรัน

การทำงานในทั้งสามภาษานี้มีความคล้ายคลึงกัน โดย:

  • เมื่อสร้างวัตถุจากคลาส จะเรียก constructor ของแต่ละคลาสเพื่อพิมพ์ข้อความ
  • การเรียกใช้เมธอด sound() จะแสดงผลตามการ override ในคลาสลูก ซึ่งแสดงถึงการทำงานของ polymorphism
  • เมื่อวัตถุถูกลบ (ด้วย garbage collection, delete หรือ del) destructors จะถูกเรียกเพื่อล้างหน่วยความจำและพิมพ์ข้อความ
